socket.io pub-sub server [zeromq](http://zeromq.org) implementation.


# Installing

```
$ npm install socket.io-zeromq-server
```

zeromq version required >= 4.0.4


# Usage

```shell
$ socket.io-zeromq-server
```


# Testing

```shell
$ make test
```
